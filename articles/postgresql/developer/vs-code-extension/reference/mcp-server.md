# MCP server reference

This page describes the Model Context Protocol (MCP) server definition providers registered by the PostgreSQL extension.

## PostgreSQL Tools MCP Server Provider

**ID:** `pgsql-tools-mcp-server-provider`

**Label:** PostgreSQL Tools MCP Server Provider

The PostgreSQL extension provides an MCP server that exposes database tools to AI assistants and language models through the Model Context Protocol.
