Use the Visual Studio Code extension for Azure DocumentDB to explore your data. You can perform core database operations including, but not limited to:

- Performing queries using a scrapbook or the query editor
- Modifying, updating, creating, and deleting documents
- Importing bulk data from other sources
- Managing databases and collections

For more information, see [How-to use Visual Studio Code extension to explore Azure DocumentDB data](../../../visual-studio-code-extension.md?pivots=api-mongodb&tabs=MongoDB).
