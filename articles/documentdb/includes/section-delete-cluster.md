When you're done with the Azure DocumentDB cluster, you can delete the Azure resources you created so you don't incur more charges.

1. In the Azure portal search bar, search for and select **Resource groups**.

    :::image type="content" source="media/quickstart-portal/search-resource-group-for-deletion.png" alt-text="Screenshot showing option for searching resource groups.":::

1. In the list, select the resource group you used for this quickstart.

    :::image type="content" source="media/quickstart-portal/click-delete-resource-group.png" alt-text="Screenshot showing resource group.":::

1. On the resource group page, select **Delete resource group**.

1. In the deletion confirmation dialog, enter the name of the resource group to confirm that you intend to delete it. Finally, select **Delete** to permanently delete the resource group.

    :::image type="content" source="media/quickstart-portal/confirm-delete-resource-group.png" alt-text="Screenshot showing delete resource group confirmation button.":::
