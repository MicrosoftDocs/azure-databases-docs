# Help and support

- For frequently asked questions, go to [FAQ](faq-citus.yml).
- [Cluster management](cluster-management.md) has details about scaling the database, isolating tenants, and dealing with failures.
- Try [Query performance tuning](performance-tuning.md) for a procedure to follow when queries are running slowly.

- For documentation of older versions (11.x and older), see [archive documentation](https://github.com/citusdata/citus_docs).

## PostgreSQL compatibility

Citus packages target specific PostgreSQL major versions. Use the following table to choose a matching version. The rows are sorted newest to oldest.

| Citus version | Compatible PostgreSQL versions | Supported Citus version |
| --- | --- | --- |
| 14.x | 18, 17, 16 | Yes |
| 13.x | 17, 16, 15 | Yes |
| 12.1 | 16, 15, 14 | Yes |
| 12.0 | 15, 14 | No |
| 11.1 - 11.3 | 15, 14, 13 | No |
| 11.0 | 14, 13 | No |
| 10.2 | 14, 13, 12 | No |
| 10.1 | 13, 12 | No |
| 10.0 | 13, 12, 11 | No |
| 9.5 | 13, 12, 11 | No |
| 9.0 - 9.4 | 12, 11 | No |
| 8.x | 11, 10 | No |
| 7.x | 10, 9.6 | No |
| 6.x | 9.6, 9.5 | No |
| 5.2 | 9.5 | No |
