# Citus API

The Citus API consists of utility functions, metadata tables and views, and configuration parameters. Use these resources to manage distributed tables, shards, and cluster operations.

## In this section

| Article | Description |
| --- | --- |
| [Citus Utility Functions](api-udf.md) | User-defined functions for distributed table management, shard operations, and cluster administration |
| [Citus Tables and Views](api-metadata.md) | Metadata tables and views for tracking shards, placements, and cluster state |
| [Configuration Reference](api-guc.md) | Configuration parameters that control Citus behavior |
